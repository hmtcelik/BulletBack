# BulletBack
>- spawning goblins left and right side randomly
>- droping knife on air randomly
>- the player can shoot
>- counting score when killing goblin
>- reloading bar top of the player
>- just survive... 

# How to Play
> Keys:
>   - Movement= Left and Right arrow key
>   - Shoot = Q 
>
